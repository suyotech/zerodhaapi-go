# Examples

These examples show package usage only. Replace placeholder credentials before running broker calls.

```text
examples/
  login/          print Kite login URL and generate access token
  place_order/    place a simple order
  positions/      fetch positions
  quotes/         fetch LTP
  historical/     fetch candles
  instruments/    download/cache instruments and find expiries/strikes
  ticker/         build ticker endpoint and messages
```
